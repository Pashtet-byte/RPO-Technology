# TaskFlow MVP

Простой локальный трекер задач для фрилансеров и малого бизнеса.

## Как запустить
1. Скачать все три файла (index.html, styles.css, script.js).
2. Открыть `index.html` в любом браузере.
3. Всё сохраняется автоматически в localStorage.

## Требования
- Современный браузер (Chrome, Firefox, Edge).
- Без интернета и установки.

## Структура проекта
- index.html — интерфейс
- styles.css — оформление
- script.js — вся бизнес-логика и localStorage
